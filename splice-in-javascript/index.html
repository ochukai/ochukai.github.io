Javascript 中的 splice 方法

splice() 方法向/从数组中添加/删除项目，然后返回被删除的项目。(会改变原始数组)

语法

参数 描述
index 必需。整数，规定添加/删除项目的位置，使用负数可从数组结尾处规定位置。
how-many 必需。要删除的项目数量。如果设置为 0，则不会删除项目。
item1, …, itemX 可选。向数组添加的新项目。
